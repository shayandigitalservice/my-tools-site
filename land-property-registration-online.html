Online Land / Property Registration 2026

Step by Step Official State Portal Guide

Online land or property registration karna ab easy aur official process hai. Ye guide aapko step by step process batayegi, jaise ki documents prepare karna, state portal se apply karna aur final registration complete karna.

Property Registration Kya Hai

Property registration ka matlab hai ki aap apna land / property official government records me register karte ho. Ye ownership proof aur legal documentation ke liye mandatory hai.

Required Documents / Details

  • Property / land details (survey number, location, etc.)
  • Owner Aadhaar / ID proof
  • Sale deed / agreement
  • Encumbrance certificate
  • Property tax receipt (if applicable)
  • Bank account details for fee payment
  • Passport size photo (owner)

Step 1 – Official State Property / Registration Portal Open Kare

Browser me apne state ka official property registration portal open kare. Examples:

  • eGrass Portal – Karnataka
  • Maharashtra SRO Portal
  • Tamil Nadu – TNRERA Portal
  • Other state portals ke liye Google me “ property registration online” search kare

Step 2 – Login / Registration

“New User / Login” option select kare. Owner details aur Aadhaar / ID verification kare:

  • Name, DOB, Gender
  • Aadhaar / PAN number verification
  • Email / mobile OTP verification
  • Set username & password for portal login

Step 3 – Property Details Fill Kare

Online application me property details enter kare:

  • Survey number / plot number
  • Property location / district / taluka
  • Previous owner / buyer details (if applicable)
  • Upload documents – sale deed, encumbrance certificate, ID proof
  • Review details carefully

Step 4 – Fee Payment

Portal me applicable registration fee pay kare:

  • Net banking / debit card / UPI / credit card available
  • Fee receipt download kare
  • Save reference number for future tracking

Step 5 – Appointment / eStamping

Kuch states me online appointment aur eStamping required hoti hai:

  • Appointment date select kare
  • Visit sub-registrar office (if required) with documents
  • Final registration / deed verification complete kare
  • Receipt aur registration certificate download kare

Step 6 – Track Registration Status

“Track Application Status” section me jaake:

  • Application number / reference number enter kare
  • Status check kare – Pending / Approved / Completed
  • Official registration certificate save kare

Official State Property Links

  • eGrass Portal – Karnataka
  • Maharashtra SRO Portal
  • Tamil Nadu – TNRERA Portal

Important Tips

  • Official state portal hi use kare
  • Documents valid aur clear ho
  • Reference number aur receipt safe rakhe
  • Deadline aur appointment dhyan me rakhe
  • Multiple properties ke liye alag application submit kare

Benefits of Online Property Registration

  • Direct government registration – secure aur legal
  • Track application status online
  • Save time aur hassle-free process
  • Digital receipts aur certificates available
  • Reduce fraud aur ownership disputes

Conclusion

Online land / property registration ab simple aur fast process hai. Upar diye steps follow karke aap apna property officially register kar sakte ho aur receipt / registration certificate secure kar sakte ho.
