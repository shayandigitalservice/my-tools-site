PF Balance & EPF Online Statement 2026

Step by Step Official EPFO Guide

PF balance aur EPF passbook online check karna ab simple aur official process hai. Ye guide aapko step by step process batayegi, jaise ki UAN login, EPFO portal, SMS aur missed call methods use karke apna balance aur statement dekh sakte ho.

EPF / PF Kya Hai

Employees’ Provident Fund (EPF) ek retirement savings scheme hai jo employer aur employee dono contribution ke saath banti hai. PF balance check karna aur passbook download karna important hai financial planning ke liye.

Required Details / Documents

  • UAN (Universal Account Number)
  • Registered mobile number linked with UAN
  • Member ID / PF account number (optional)
  • Email ID (optional for notification)

Step 1 – Official EPFO Portal Open Kare

Browser me EPFO Member e-Sewa Portal open kare.

Step 2 – Login via UAN

“Login” option select kare aur UAN aur password enter kare:

  • UAN number enter kare
  • Password enter kare
  • Captcha / OTP verification
  • Login successfully

Step 3 – PF Passbook / Balance Check

  • Dashboard me “View Passbook / Account Statement” option select kare
  • Employer aur contribution details dekh sakte hain
  • Download / print passbook for record
  • Recent transactions aur balance check kare

Step 4 – SMS / Missed Call Method

Kuch employees ke liye mobile se bhi PF balance check possible hai:

  • SMS: Type “EPFOHO UAN” and send to 7738299899
  • Missed Call: Give missed call to 011-22901406 (for registered mobile)
  • Receive instant balance details via SMS

Step 5 – UAN / Mobile App Method

EPFO ka official mobile app bhi available hai:

  • EPF Mobile App install kare (Android / iOS)
  • Login with UAN aur password
  • Passbook aur balance check kare real-time
  • Screenshot save kare future reference ke liye

Official EPFO Links

  • EPFO Member e-Sewa Portal
  • EPFO Official Website
  • EPF Mobile App (Android)
  • EPF Mobile App (iOS)

Important Tips

  • Official EPFO portal hi use kare
  • UAN aur registered mobile number active ho
  • Passbook regularly check kare
  • OTP aur password secure rakhe
  • Employer aur employee contribution details confirm kare

Benefits of Online PF / EPF Check

  • Real-time balance aur transaction check
  • Download passbook for records
  • Secure aur official method
  • No branch visit required
  • Financial planning easy

Conclusion

PF balance aur EPF online statement check karna ab fast aur simple hai. UAN login, mobile app, SMS aur missed call methods use karke aap apna balance aur passbook easily dekh sakte ho aur official records save kar sakte ho.
