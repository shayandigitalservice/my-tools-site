UAN Forgot Password Reset Online 2026

Step by Step Official EPFO Password Recovery Guide

Agar aap apna UAN password bhool gaye ho, to tension lene ki jarurat nahi hai. EPFO portal ke through aap easily apna password reset kar sakte ho. Is guide me hum aapko step by step batayenge ki kaise UAN forgot password reset kare.

UAN Password Reset Kya Hai

UAN (Universal Account Number) EPF account ke liye ek unique ID hoti hai. Agar aap password bhool jaate ho, to “Forgot Password” option ke through naya password set kar sakte ho.

Required Details

  • UAN Number
  • Registered mobile number (UAN ke sath linked)
  • Aadhaar / PAN details (optional verification ke liye)
  • Internet connection aur mobile OTP receive karne ke liye active SIM

Step 1 – Official EPFO Portal Open Kare

Browser me EPFO Member e-Sewa Portal open kare.

Step 2 – Forgot Password Option Select Kare

  • Login page par “Forgot Password” option click kare
  • UAN number enter kare
  • Captcha verification complete kare

Step 3 – Mobile Number Verification

  • Registered mobile number display hoga
  • “Yes” select kare agar number active hai
  • OTP receive kare aur enter kare

Step 4 – Aadhaar / PAN Verification (Optional)

  • Aadhaar ya PAN details verify karne ke liye option aa sakta hai
  • Details enter kare aur proceed kare

Step 5 – New Password Set Kare

  • Naya password enter kare (strong password use kare)
  • Password confirm kare
  • Submit button click kare
  • Password successfully reset ho jayega

Step 6 – Login with New Password

  • Back to login page jaye
  • UAN aur new password enter kare
  • Dashboard access kare successfully

Official EPFO Links

  • EPFO Member Portal

  • EPF Employee Services

Important Tips

  • Registered mobile number active hona chahiye
  • Strong password use kare (numbers + letters + symbols)
  • OTP kisi ke sath share na kare
  • Official EPFO portal hi use kare
  • Password safe jagah note kare

Common Problems & Solutions

  • Mobile number change ho gaya: EPFO helpdesk contact kare
  • OTP receive nahi ho raha: Network check kare ya retry kare
  • UAN inactive hai: Pehle UAN activate kare
  • Password error: Strong password rules follow kare

Benefits of Online UAN Password Reset

  • Instant password recovery
  • No office visit required
  • Secure OTP verification system
  • Easy login access restore hota hai
  • Free aur official process

Conclusion

UAN password reset karna ab fast aur easy hai. Upar diye steps follow karke aap apna password recover kar sakte ho aur EPFO services ka use dobara kar sakte ho bina kisi problem ke.
