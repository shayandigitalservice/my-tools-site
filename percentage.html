What is Percentage?

A percentage is a number expressed as a fraction of 100. It is represented using the symbol (%). Percentages help compare values and understand proportions easily.

For example:

  • 50% means 50 out of 100
  • 25% means 25 out of 100
  • 100% means the whole value

Percentage Formula

1. Basic Percentage Formula

Percentage = (Value / Total Value) × 100

2. Percentage of a Number

Result = (Percentage × Number) / 100

3. Percentage Increase Formula

Increase % = ((New Value - Old Value) / Old Value) × 100

4. Percentage Decrease Formula

Decrease % = ((Old Value - New Value) / Old Value) × 100
