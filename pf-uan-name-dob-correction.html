PF UAN Name / DOB Correction 2026

Step by Step Guide for Employees

Important:
EPF UAN me Name / DOB correction sirf official EPF portal ya employer ke through hi hota hai.

PF UAN Correction Kya Hai?

PF UAN me Name / DOB correction ka matlab hai apne EPF account me jo employee details hai (Name, Date of Birth) unko sahi karwana. Ye important hai kyunki claim processing ke liye details match honi chahiye.

Correction Karne Ke Liye Required Documents

  • Scanned Aadhaar / PAN / Passport (as per required proof)
  • Employee ID / UAN Number
  • Official request / signed application (optional, employer ke liye)

Step by Step – UAN Name / DOB Correction

Step 1 – EPF Portal Login Kare

👉 Official EPF Member Portal

Step 2 – UAN Number Enter Kare

UAN number aur password enter kare aur login kare.

Step 3 – Profile / KYC Section Open Kare

Profile section me jaye aur Name / DOB correction option select kare.

Step 4 – Correct Details Enter Kare

Correct Name / DOB type kare aur supporting documents attach kare.

Step 5 – Submit Request

Submit karne ke baad request employer ke paas jaati hai approval ke liye.

Step 6 – Employer Verification

Employer verify karega details aur approve karega.

Step 7 – EPFO Approval

Approval ke baad PF portal me details automatically update ho jaati hai.

Important Tips

  • Documents clearly scan kare
  • Official EPF portal hi use kare
  • Employer ko inform karna optional but recommended
  • Login credentials safe rakhe

Conclusion

PF UAN Name / DOB correction ka process simple hai agar proper documents aur official portal follow kare. Upar diye gaye link se turant login karke request submit kar sakte ho.
