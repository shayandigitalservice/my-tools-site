2. Technical Core: Mathematical Logic of Triangulation via Diagonal Boundaries

Ek irregular quadrilateral (4 corners wali asamatal zameen) ka real surface configuration tabhi calculate ho sakta hai jab use kisi ek corner se opposite corner tak ki line draw karke do separate triangles me segment kiya jaye. Is line ko geometric language me **Diagonal (କର୍ଣ୍ଣ - E)** kaha jata hai. Jab hamare paas diagonal mil jata hai, tab hamara calculator use do bhagon me baant deta hai: Triangle 1 (Side A, Side B, Diagonal E) aur Triangle 2 (Side C, Side D, Diagonal E).

Donon triangles ka calculation individually **Heron's Formula** ke rigorous steps se guzarta hai: $$\text{Semi-perimeter } (s) = \frac{a + b + \text{diag}}{2}$$ $$\text{Area} = \sqrt{s(s-a)(s-b)(s-\text{diag})}$$ In donon separate areas ko sum up karne par sateek absolute square feet nikal kar aata hai. System is computed values ko bina kisi manual errors ke instant automatic logic se Odisha state ke custom revenue systems jaise ki **Guntha, Decimal (Dismil), Bigha, Acre aur Hectare** me precision analytics ke sath map pe plot kar deta hai.
